## 🚀 Comandos Essenciais

```bash
# Setup
yarn install

# Desenvolvimento
yarn dev              # Start MFE Root
yarn build            # Build tudo
yarn test             # Rodar testes
yarn lint             # Lint do código
yarn type-check       # TypeScript check

# Workspace específico
yarn workspace @call-center-platform/mfe-shell dev
yarn workspace @call-center-platform/shared-ui test
```

## 🧩 Shared Packages (O que cada um oferece)

| Package | Oferece | Exemplo |
|---------|---------|---------|
| **shared-ui** | Componentes React | Button, Card, Badge |
| **shared-utils** | Funções úteis | formatDate, delay, fetch |
| **shared-types** | Types/Interfaces | User, Call, FeatureToggle |

---

## 🔗 Como Importar Coisas

### De um shared package
```typescript
import { Button } from '@call-center-platform/shared-ui';
import { formatDate } from '@call-center-platform/shared-utils';
import type { User } from '@call-center-platform/shared-types';
```

### De mfe-shell (global state)
```typescript
import { useGlobalStore } from '@call-center-platform/mfe-shell';
```
